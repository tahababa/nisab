01What is Nisab Al Zakat?

Nisab Al Zakat is a free, open JSON API for the Nisab threshold — the minimum wealth at which Zakat becomes obligatory. Values are calculated in real time from live gold and silver prices, across all four major schools of Islamic jurisprudence, in 27 currencies, updated 6 times every day.

No authentication. No rate limits. No registration. Just a clean JSON endpoint you can fetch from any app, website, or automation. Every snapshot is permanently archived, building a growing historical record of Nisab thresholds over time.

02The Endpoints

Today (live)
curl https://nisab.tahababa.com/nisab.json
Specific date
curl https://nisab.tahababa.com/history/2026-03-15.json
History index
curl https://nisab.tahababa.com/history/index.json
// → { "count": 365, "dates": ["2026-03-15", "2026-03-14", ...] }
JavaScript — get one value
const { nisab } = await fetch('/nisab.json').then(r => r.json());

// Hanafi gold nisab in GBP today
nisab.hanafi.gold.values.GBP   // → 6842.11

// Maliki silver nisab in SAR
nisab.maliki.silver.values.SAR  // → 2218.40

03Historical Archive

Every snapshot is permanently committed to the GitHub repository. 6 snapshots per day means ~2,190 records per year — a full, queryable price history of Nisab thresholds building itself automatically.
